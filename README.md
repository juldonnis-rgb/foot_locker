# ⚽ FreeLockickX — Interface Instant Gagnant

**FreeLockickX** est un projet d'interface web responsive conçu pour un jeu de type "Instant Gagnant" sur le thème du football. L'utilisateur est mis au défi de marquer un coup-franc décisif à la dernière minute du temps réglementaire pour tenter de remporter un prix.

---

## 🎯 Concept du Projet

> "C'est la fin du temps réglementaire, sauve ton équipe avec ce coup-franc !"
> 

L'objectif était de créer une interface **"Glassmorphism"** épurée qui met l'accent sur l'action de jeu tout en restant parfaitement lisible sur ordinateur, tablette et mobile.

### Points clés de l'interface :

- **Navigation fluide :** Un header moderne avec un logo stylisé.
- **Zone de jeu (Game Zone) :** Un conteneur central dédié à l'interaction.
- **Écran de feedback :** Une zone de texte dynamique pour guider le joueur.
- **Call-to-Action (CTA) :** Un bouton d'engagement clair pour lancer la partie.

## 📂 Structure des fichiers

Le projet est organisé de manière minimaliste pour une performance optimale :

- `index.html` : Contenu et structure de la page.
- `style.css` : Design, typographie et mise en page responsive.
- `assets/` : (À venir) Dossier pour les icônes de ballons ou images de stade.
